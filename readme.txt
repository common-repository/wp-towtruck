TowTruck is a tool from Mozilla Labs that lets users communicate, co-author, co-browse and guide each other. Visit <a href="https://towtruck.mozillalabs.com/" target="_blank">Mozilla TowTruck official site</a> for more informations.== Installation ==1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/2. Go into the WordPress admin interface and activate the plugin3. Go to the options page and activate TowTruck button== Frequently Asked Questions ==No questions have been asked.== Screenshots ==No screenshots are available.== Changelog === 1.1 =* Custom button styles* Admin interface restyling* Code optimizations and comments= 1.0 =* First release
